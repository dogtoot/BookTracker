## Book Tracker
A book management application designed for the Android Development class at Volunteer State Community College.

## Features
- Track books you're reading, have read, or want to read
- Scan barcodes to add books instantly
- Mark favourites

### Requirements
- Android Studio Koala or newer.
- Android SDK version: 35
- Minimum SDK version: 24

### Installation
1. Clone the repo
`git clone https://github.com/dogtoot/BookTracker.git`
2. Open in Android Studio
3. Build and run either on an emulator, or your device.
